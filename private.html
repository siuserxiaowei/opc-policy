策略一：南沙/前海设子公司，企税降到15%

方案：在南沙设立子公司，承接对外业务
潜在节省：企税从25%→15%，年利润100万可省 10万

操作路径

  1. 在南沙注册一家子公司（一人有限公司），最快0.5天拿照
  2. 子公司主营业务定为"AI技术服务/软件开发"（属于鼓励类产业）
  3. 越秀母公司签研发/方案设计合同给南沙子公司执行
  4. 对外客户的合同由南沙子公司签署和开票
  5. 利润留在南沙子公司，享受15%企税
  6. 南沙子公司还可叠加小微5%优惠（年利润≤300万部分）

风险提示：必须有实质性运营（实际办公、员工、业务往来），不能是空壳。南沙有OPC专窗可以帮你搞定注册。税务局会看关联交易定价是否合理。

重要：南沙15%政策执行到2026年12月31日，是否续期未知。且年利润≤300万时小微5%比南沙15%更划算，建议利润超300万再设子公司。

方案B：前海（深圳）子公司
同样15%企税，适合有深圳客户的情况
  1. 前海15%企税已扩至全域，AI/信息服务直接适用
  2. 主营收入占比≥60%即可，不需要额外申请
  3. 报税时自行申报15%税率
  4. 如果有港澳客户，前海还有跨境便利

注意：前海注册成本比南沙高，但深圳客户资源更丰富。

策略二：研发费用加计扣除

你的AI语音机器人研发投入可以享受120%加计扣除
假设年研发投入50万，额外扣除 60万（50万×120%），直接减少应纳税所得额

哪些费用算研发

  • 程序员工资（包括你自己如果参与开发）
  • 服务器/算力/云服务费用
  • AI模型训练的Token消耗
  • 软件工具采购（IDE、API调用等）
  • 硬件样机材料费（ESP32、麦克风阵列等）
  • 测试设备折旧
  • 外包的技术开发费（有合同和发票）

申报流程

  1. 建立研发项目台账（项目名称、起止时间、人员、费用）
  2. 研发费用单独核算（最好有独立的研发费用科目）
  3. 年度汇算清缴时填报《研发费用加计扣除优惠明细表》
  4. 留存备查资料：立项文件、研发人员名单、费用明细、成果说明
  5. 不需要事先审批，自行申报享受

关键：必须是"新技术/新产品/新工艺"的研发，日常维护不算。AI语音机器人的算法优化、新功能开发都算。

策略三：软件产品增值税即征即退

软件产品实际税负超过3%的部分即征即退
一般纳税人13%增值税，实际只交 3%，退回 10%

条件

  • 取得软件产品登记证书（软著即可）
  • 必须是一般纳税人（年收入超500万或主动申请转为一般纳税人）
  • 销售自主开发的软件产品
  • 你的AI语音对话机器人如果作为软件产品销售，完全适用

操作

  1. 先申请软件著作权登记（已有最好）
  2. 做软件产品检测（第三方检测机构）
  3. 到税务局备案软件产品
  4. 每月开具软件产品发票后，申请增值税即征即退

注意：嵌入式软件也适用（硬件+软件一体的产品），但需要分别核算硬件和软件的价格。

策略四：高新技术企业认定

认定后企税从25%降到15%，叠加各区奖励
企税减免 + 越秀区奖励（预计10-20万）+ 研发补助

认定条件

  • 成立满1年以上
  • 拥有核心知识产权（软著即可，建议申请3-5个）
  • 主要产品属于国家重点支持的高新技术领域（AI/电子信息/智能制造都算）
  • 科技人员占比≥10%（5人里至少1人是技术人员，你自己算）
  • 近三年研发费用占比：收入<5000万的，研发占比≥5%
  • 高新收入占比≥60%

各区奖励叠加

越秀区首次认定预计 10-20万
黄埔区研发费补贴最高 1200万
南沙区市外落户 50万
天河区租金补贴最高 1000万/年
增城区首次认定 20万 + 纳入四上额外 50万

建议：现在就开始准备，先把软著数量攒到3-5个，研发台账建起来，明年申报。

策略五：资金效率优化

业务拆分 + 多主体架构

推荐架构

主体注册地功能税率
越秀母公司越秀品牌持有、研发中心小微5% / 高企15%
南沙子公司南沙对外签约、开票、收款15%鼓励类
个体户（可选）灵活咨询服务、个人IP变现核定~5%

资金流转

  1. 客户合同由南沙子公司签署 → 享受15%企税
  2. 南沙子公司向越秀母公司支付研发服务费 → 合理转移利润
  3. 越秀母公司研发费用加计扣除120% → 减少应税所得
  4. 个人咨询/培训/OPC网站收入走个体户 → 核定征收~5%
  5. 利润留在公司不分红 → 递延个税20%

关键：关联交易定价要合理（参考市场价），转让定价文档要准备好。金额大了建议找税务师出方案。
