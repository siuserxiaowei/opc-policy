# 广州市通用创业补贴 + OPC创新工坊

OPC 创业者注册为个体工商户或一人有限公司后，与小微企业适用相同的创业扶持政策。

---

## 1. 一次性创业资助

| 项目 | 内容 |
|---|---|
| 补贴金额 | 10,000 元（一次性） |
| 申请条件 | 法人需属以下之一：高校/中职/技校学生（在校及毕业5年内）、留学回国人员、军转干部/退役军人、登记失业人员、就业困难人员、本省脱贫人口、返乡创业人员 |
| 企业要求 | 登记注册满6个月，申请前连续3个月有在职员工正常缴纳社保，未被列入经营异常名录 |
| 年龄限制 | 女性≤55岁，男性≤60岁 |
| 申请方式 | 登录[广东政务服务网](https://www.gdzwfw.gov.cn/)，以法人身份搜索「一次性创业资助」 |

**来源**：[广州本地宝](http://gz.bendibao.com/life/20191031/255749.shtml)

## 2. 创业租金补贴

| 项目 | 内容 |
|---|---|
| 补贴金额 | 每年最高 6,000 元（珠三角地区） |
| 补贴期限 | 累计不超过 3 年 |
| 申请条件 | 同一次性创业资助 |

## 3. 创业带动就业补贴

| 项目 | 内容 |
|---|---|
| 招用3人以下 | 每人 2,000 元 |
| 招用3人以上 | 每增加1人补贴 3,000 元，累计最高 30,000 元 |
| 要求 | 招用毕业两年内高校毕业生，可返还单位缴纳社保部分 |

## 4. 港澳青年"四免"扶持

| 项目 | 内容 |
|---|---|
| 补贴内容 | 免租金/水电/物业/网络 |
| 最高金额 | 50万元 |
| 期限 | 最长3年 |

## 6. 广州OPC创新工坊（市级，天河）

| 项目 | 内容 |
|---|---|
| 地址 | 广州市天河区珠江新城华利路61号 广州市政务服务中心五楼509室 |
| 服务时间 | 工作日 9:00-12:00, 13:00-17:00 |
| 服务内容 | 国产开源AI智能体框架工具（含四层安全防护）+ 基础Token流量包 + 一对一技术指导 |
| 技术支持 | 广州数据集团 + 三大运营商 |
| 预装服务 | 法务、财税辅助技能 |
| 申请对象 | 广州市注册的个人独资企业、小微企业及个人创业者 |
| 预约方式 | 通过[穗@i企平台](https://qyfw.gzonline.gov.cn/qyfw/home)预约，1个工作日审核 |
