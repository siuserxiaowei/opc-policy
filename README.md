# OPC 全国政策导航

> 一键匹配你能申请的 OPC（一人公司）补贴和政策

**在线访问：** https://siuserxiaowei.github.io/guangzhou-opc-policy/

## 功能

- **用户匹配系统** — 填写公司情况，自动匹配可申报的政策
- **结构化政策数据库** — 每条政策拆分为标准字段，方便检索和匹配
- **11座城市覆盖** — 广州、深圳、北京、上海、杭州、成都、武汉、合肥、南京、长沙、重庆
- **广州专版详情页** — 含倒计时、搜索、区域筛选、材料清单、深浅主题等交互
- **匹配结果导出** — 一键导出匹配报告
