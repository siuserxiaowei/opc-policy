# 海珠区（琶洲）— 政策力度最大

## 《广州市海珠区人工智能OPC创新发展措施（试行）》（"OPC十条"）

**状态**：征求意见稿（征求意见期：2026年3月23日 - 4月23日）
**发文单位**：海珠区人工智能发展局
**征求意见邮箱**：pzszjj@yeah.net
**邮寄地址**：海珠区赤洲街宝珠广场1001
**政策原文**：[海珠区政府征求意见页](https://www.haizhu.gov.cn/hdjlpt/yjzj/answer/49445)
**媒体报道**：[网易·最高支持500万](https://m.163.com/dy/article/KOG5K38T05129QAF.html) | [南方财经](https://www.sfccn.com/2026/3-19/yMMDE0NzNfMjExODcyMQ.html)

**适用范围**：在琶洲核心区和环海珠湿地创芯价值圈范围内的 OPC 企业，人员规模一般不超过10人，特别优秀经认定可放宽到20人。
**地理范围**：西起广州大道连东晓南路，东临珠江航道，北至珠江前航道，南至珠江后航道。

## 核心补贴条款

| 条款 | 支持方向 | 补贴金额 |
|---|---|---|
| 第3条 | 智能体服务专区（工位免租） | 每年最高 **12个月工位费全免** |
| 第4条 | OPC特色孵化空间建设（运营主体） | 每年最高 **150万元** |
| 第5条 | 琶洲模方SOPC加速器 | 微种子+成长期权+投贷联动全周期资本支持 |
| 第6条 | 算力券（50%补贴） | 最高 **20万元/年** |
| 第6条 | 数据券（30%补贴） | 最高 **10万元/年** |
| 第6条 | 流量券 | 大厂生态平台定向投放（金额待定） |
| 第7条 | 多元化生态建设 | 最高 **50万元** |
| 第8条 | 场景资源开放（项目投入30%） | 最高 **500万元** |
| 第9条 | 大模型备案奖励 | 非规上 **50万** / 规上 **100万** |
| 第9条 | 算法备案奖励 | 非规上 **15万** / 规上 **20万** |
| 第10条 | 金融综合服务 | OPC专项信贷产品 |
| 第11条 | 政务一站式 | 工商注册/财税/资质全流程代办 |
| 第12条 | 人才保障 | 纳入AI高端人才库 |

**资金承担**：第4、8、9条由广州市和海珠区按1:1共同承担。

> 注意：该政策仍在征求意见阶段（截止4月23日），正式落地可能需要1-2个月。

## 琶洲模方（SOPC超级个体加速器）⭐ 已验证运营中

| 项目 | 内容 |
|---|---|
| 性质 | 粤港澳大湾区首个大模型孵化空间 |
| 开业时间 | 2024年4月（已运营近2年） |
| 运营方 | 海珠城发集团 + 清智资本（清华大学智能产业研究院AIR支持） |
| 模式 | "北京技术策源 + 广州产业落地"双城联动 |

| 入驻优惠 | 半年以上免租期 + 算力/数据全方位支持 |
| 目标 | 2-3年聚集100+AI创业项目，培育10家独角兽 |
| 入驻方式 | ① 参加琶洲算法大赛OPC主题赛（以赛代评）② 提交BP评审 |
