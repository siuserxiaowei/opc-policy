## 核心补贴条款（已验证金额）

| 支持方向 | 补贴金额 | 申请条件 |
|---|---|---|
| 通用大模型算力补助 | 实际投入30%，单主体最高 **300万元** | 购买或租用区内智能算力 |
| 垂直大模型算力补助 | 实际投入30%，单主体最高 **100万元** | 同上 |
| 小微企业云服务券/大模型券 | **2000元/张**，共2000张 | 小微企业 |
| 大模型备案奖励 | 首次国家级备案最高 **100万元** | 模型落地应用有示范性 |
| 算法备案奖励 | 最高 **20万元** | 国家级深度合成服务算法备案 |
| 场景应用示范 | 总额最高 **1000万元**（≤30个项目） | AI赋能新型工业化应用 |
| 可信数据空间 | 获上级认可最高 **100万元** | 构建可信数据空间 |
| 开源社区建设 | 每家最高 **100万元**（≤10家） | 下载量、性能、影响力综合评定 |

> ⚠️ 原文档写"算力单主体最高500万"有误，经核实通用大模型最高300万、垂直大模型最高100万。

## 鱼珠·元启 OPC生态社区 ⭐ 已验证

| 项目 | 内容 |
|---|---|
| 性质 | 广州开发区首个OPC生态社区 |
| 签约时间 | 2026年3月20日 |
| 签约方 | 鱼珠街道 + 广州港集团 + 鼎泰控股 + 科学城数字科技集团 + 阿里云 |
| 地址 | 海丝城鱼珠片区、港航中心、南方智媒大厦 |
| 首批类型 | 综合类 OPC + 游戏产业创作类 OPC |
| 免租政策 | 优质项目最长 **3年免租**，后续2年租金减半 |
| 算力支持 | 接入"黄埔一号"智算集群超级算力 |
| TOKEN激励 | 入驻即获3个月免费TOKEN，注册企业享1年TOKEN服务 |
| 入驻联系 | 鼎泰控股 / 鱼珠运营中心 |
