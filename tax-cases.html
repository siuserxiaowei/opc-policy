案例一：拒签50万单子的会计

跟你直接相关
场景：小规模纳税人，客户要签50万大单

客户：签个50万的单子吧？

上海会计：不能签。我们是小规模纳税人，接了这单年收入超500万，税务局强制转一般纳税人，增值税从1%升到13%。供应商没进项票，成本无法抵扣，利润全变税基，所得税直接顶格25%。

老板：那买辆50万的车呢？

上海会计：买！车可一次性计入成本扣除，账面利润从340万降到290万，所得税从85万降到15万，省70万。这叫资产配置。

核心知识点

1. 小规模 vs 一般纳税人的生死线 = 年开票500万

小规模增值税1%（2026年优惠税率），一般纳税人13%。差了12倍。

2. 没有进项票的一般纳税人是灾难

如果你的上游（供应商、分包方）给不了增值税专用发票，你转成一般纳税人后13%全是自己扛。

3. 固定资产一次性扣除是合法的"利润压缩器"

国家政策允许500万以下的设备/车辆一次性计入当期成本（不需要分年折旧），直接减少当年应纳税所得额。

跟你的关系

你现在是小规模纳税人，做AI语音机器人方案。务必盯紧年开票额：

- 接近500万时，要么控制开票节奏（推迟到下一个自然年），要么提前规划好进项票来源

- 如果某个大单会让你越线，可以拆分成两个自然年的合同

- 年底利润太高？买设备（服务器、开发用电脑、测试设备）一次性扣除，合法省税

- 你的AI硬件（ESP32开发板、麦克风阵列、测试样机）全都算

案例二：公司注销的"最后一笔账"

段子
场景：公司要注销，账上50万怎么拿出来

老板：我账上还有50万，你转给我就行。

上海会计：直接转？50万分红要交20%个税，到手只剩40万。

老板：那怎么办？

上海会计：先"开除"你自己，拿N+1劳动补偿金。补偿金在当地年均工资3倍以内免税。剩下的部分再分红。

核心知识点

1. 分红税20%是硬伤

公司赚的钱，交完25%企税，再分红给个人还要交20%个税。100万利润最终到手只有60万。

2. 劳动补偿金有免税额度

N+1补偿金中，不超过当地上年职工年均工资3倍的部分免征个税。广州2025年标准约13万/年，3倍就是39万免税。

3. 这不是偷税，是税目转换

"劳动补偿"和"股东分红"是两个不同税目，税率天差地别。合法利用不同税目的税率差异叫税务筹划。

跟你的关系

你现在不需要注销，但要记住两件事：

- 尽量不分红。利润留在公司里，用于业务扩展、买设备、发工资，都比直接分红划算

- 如果需要从公司拿钱，优先用工资+合理报销的方式，工资有5000/月免征额+专项扣除

- 万一哪天公司不做了，提前规划退出路径，别等到注销那天才想这个问题

案例三：开除老板娘平账

高频踩坑
场景：老板从自己公司借了30万，一直没还

上海会计：老板，你从公司借的30万还了吗？

老板：我的公司我的钱，还什么还？

上海会计：公私不分，这是挪用公款。而且借款超过一个纳税年度不归还，税务局视同分红，要交20%个税。30万就是6万的税。

核心知识点

股东借款超期 = 视同分红，必须交20%个税

依据：财税[2003]158号文，个人投资者从公司借款，在纳税年度终了后既不归还又未用于经营的，视为分红。

正确做法

- 年底前归还，下一年再借（"借新还旧"，但不能太明显）

- 有明确用途的备用金借支，保留用途证明

- 或者干脆通过工资、报销等正规渠道拿钱

跟你的关系

你是一人有限公司，这个坑最容易踩：

- 公司账户的钱不能随便转到个人卡，每一笔都要有名目（工资/报销/借款）

- 如果有借款，年底前必须还，否则被视同分红交20%

- 一人公司还有个特殊风险：如果不能证明个人财产和公司财产独立，可能被"刺破公司面纱"，承担无限责任

- 建议：每年做一次审计报告，证明公私分离

案例四：财务人员工资进"财务费用"？

冷知识
场景：新会计按字面意思归类工资科目

新会计：车间工人工资进制造费用，销售人员进销售费用，管理人员进管理费用。

上海会计：那财务人员工资呢？

新会计：进财务费用啊！

上海会计：哈哈，"财务费用"不是放财务人员工资的，是放利息、手续费、汇兑损益的。财务人员工资进"管理费用"。

核心知识点

科目分错会影响报表和税务申报

- 管理费用：行政人员、财务人员的工资

- 销售费用：销售人员的工资

- 财务费用：利息支出、银行手续费、汇兑损益（跟"人"无关）

为什么重要？

科目归错会让研发费用加计扣除、高企认定时的费用比例计算出问题，直接影响你能不能拿到优惠。

案例五：一张发票差出20万税

跟你直接相关
场景：AI方案商给客户报价100万，发票怎么开？

老板：客户要100万的AI语音方案，我开一张100万的"技术服务费"发票行吗？

会计：等等。这100万里面，硬件（设备、模组）占多少，软件和服务占多少？

老板：硬件大概40万，软件开发和部署60万。

会计：那必须分开开。硬件开"电子设备"13%，软件开"信息技术服务"6%（小规模1%）。全开成13%的话，60万的软件部分白多交了7%，就是4.2万。

会计：更好的做法——如果软件部分能拿到软著，走"软件产品销售"，一般纳税人还能即征即退到3%。

核心知识点

混合销售 vs 兼营：开票方式决定税率

- 硬件+软件一起卖，如果不分开核算，全部按最高税率13%缴税

- 分开核算后，硬件13%，技术服务6%，软件产品即征即退实际3%

AI方案商的最优开票结构

- 硬件部分：货物销售13%（小规模1%）

- 软件产品部分：有软著可走即征即退（实际3%）

- 技术服务部分：信息技术服务6%（小规模1%）

- 部署运维部分：信息技术服务6%（小规模1%）

跟你的关系

你做AI语音机器人方案，每一单都涉及硬件+软件+服务，这是最容易多交税的地方：

- 跟客户签合同时，把"硬件采购"、"软件授权"、"技术服务"拆开写

- 对应开三种发票，各走各的税率

- 软件部分去申请软著，未来转一般纳税人后可以即征即退

- 一年做10单，光这一个操作可能省5-10万

案例六：5个人的社保怎么交最划算

跟你直接相关
场景：小公司5个人交社保，全按最低基数？

老板：社保全按最低基数交，省钱。

会计：现在不行了。2025年起社保入税，税务局直接拿你的工资申报数据和社保基数比对。工资报8000、社保按4000交，一对就穿了。

老板：那怎么办？

会计：合法的方式是调整薪酬结构。基本工资定合理水平，其他通过合规的福利形式发放——比如通讯补贴、交通补贴、差旅补贴，这些在标准内不计入社保基数。

核心知识点

社保入税后，基数造假风险极大

- 税务局可以直接交叉比对个税申报和社保基数

- 被查到补缴+滞纳金+罚款，可能比省下的还多

合规的薪酬优化方式

- 合理设定基本工资（不低于当地最低工资标准）

- 通讯补贴（广州标准内免税，约300-500/月）

- 差旅补贴（实报实销部分不计入）

- 年终奖单独计税（可以选择并入综合所得或单独计税，哪个低选哪个）

跟你的关系

你公司5个人交社保，每个月社保支出不小：

- 广州2025年社保最低基数约4588/月，公司部分约占工资的30%+

- 5个人一年社保公司部分大约 8-12万（取决于基数）

- 如果有人可以走兼职/劳务合同而非全职雇佣，可以省社保（但要确保业务真实性）

- 技术外包给个体户，走B2B服务费发票，也是合规降成本的方式

案例七：咨询费100万，交多少税？

节税利器
场景：同样100万收入，不同主体交的税天差地别
收入方式税种税率100万到手
个人直接收劳务报酬20%-40%约 60万
公司收→分红企税25%+分红20%综合40%约 60万
小微公司收→分红企税5%+分红20%综合24%约 76万
个体户（核定征收）核定应税所得率综合约5%约 95万

核心知识点

个体户核定征收是合法的节税王炸

- 核定征收：税务局不看你的实际利润，按收入的一个比例（通常5%-10%）核定应税所得

- 100万收入，核定应税所得率10%，应税所得=10万，个税约5000-1万

- 加上增值税1%（小规模）≈1万，综合税负约2-5%

注意事项

- 各地核定政策不同，广州目前部分区还能核定

- 业务必须真实，不能是纯粹的"开票工具"

- 如果只给一个甲方开票，容易被认定为虚开

跟你的关系

你的策略五已经规划了个体户作为多主体架构的一部分：

- OPC网站收入（咨询、广告）→ 走个体户，核定征收

- 个人IP变现（培训、课程、自媒体）→ 走个体户

- 注意：个体户必须有2-3个不同客户，不能只服务你自己的公司

案例八：Token费也能抵税？

场景：AI公司每月花3万调用大模型API，这笔钱能不能少交税？

会计：超出的10万按25%交，300万以内还是5%。但你可以在年底前把利润压下来。

老板：怎么压？

会计：几个合法方式——

年底利润调节工具箱（全部合法）

1. 提前采购设备 — 500万以下一次性扣除

6. 加速计提坏账准备 — 有超期应收账款的可以计提

注意：所有操作必须在12月31日之前完成，且有真实业务和发票支撑。纯粹为了避税的虚假交易违法。

如果你今年利润接近300万的线：

- 提前囤一批ESP32模组、服务器、办公设备

- 给自己和员工发合理的年终奖

案例十一：800亩地的财税术 — 大哥的真实操作

真实案例
背景：2011-2014年，无锡开发区，制造业老板，做产品加工接外单

大哥：赚了钱不能放账上，放着就是利润，利润就要交税。

大哥：买地。开发区的地便宜，政府还求着你买。利润变成地，成本上去了，税就下来了。

大哥：地拿到手，拿去银行抵押，贷款又出来了。手上的现金没少，但账上的利润没了。

大哥：贷款还要还利息？利息也是成本，又抵一轮税。

大哥：我这800亩算少的了，旁边那几个老板都这么搞。

操作链条拆解

利润（要交25%企税）
  ↓ 买地建厂房
固定资产（土地摊销 + 厂房折旧 = 成本↑ 利润↓ 税↓）
  ↓ 抵押给银行
贷款现金（贷款不是收入，不交税，钱又回来了）
  ↓ 还贷款
利息支出（财务费用，再抵一轮税）
  ↓ 同时
土地升值（资产增值，但不卖就不触发税）

为什么这套操作在当时几乎无风险

地价便宜开发区招商引资，工业用地远低于市场价，有些象征性收费
土地升值长三角工业地皮年年涨，买了就是赚
银行配合实体制造业有地有厂房，银行放贷宽松，抵押率50%-70%
政府鼓励你买地建厂解决就业，政府还给你补贴和税收返还

核心财税知识点

1. 利润→资产 = 合法的利润压缩

企业购买的固定资产（土地、厂房、设备），通过折旧/摊销计入成本，直接减少应纳税所得额。这是税法鼓励的——国家希望企业把钱花在扩大再生产上。

2. 资产抵押贷款 ≠ 收入

贷款是负债，不是收入，不产生纳税义务。花出去的钱通过贷款"回收"，但税已经少交了。

3. 贷款利息 = 财务费用 = 再抵税

企业经营性贷款的利息支出，可以税前扣除。相当于银行帮你再省一轮。

4. 资产不卖就不触发增值税

土地涨了10倍，只要不卖，就不用交增值税和所得税。长期持有 = 无限递延纳税。

缩小版：你能用的等价操作

你没有买地的体量，但底层逻辑完全一样——利润变资产，资产抵税，现金回流：

大哥的做法你的等价操作
利润 → 买800亩地利润 → 买设备/服务器/预付云服务年费
土地折旧摊销抵税500万以下设备一次性扣除抵税
地抵押 → 银行贷款设备融资租赁 / 经营贷（未来规模大了可用）
贷款利息 → 财务费用抵税经营贷利息 → 同样可以抵税
地升值但不卖设备持续使用，残值为0但还能用

记住这个公式：赚到的钱，先花在能抵税的地方，再想办法让现金回来。

案例十二：薇娅13.41亿罚单 — 个体户避税的红线

反面教材
2021年12月，头部主播薇娅（黄薇）被追缴税款、加收滞纳金并处罚款共计13.41亿元

她怎么操作的

1. 设立多个个人独资企业和合伙企业

在多个税收洼地（上海崇明、江苏等）注册了十几个个体户/个独企业，享受核定征收。

2. 把个人劳务报酬转化为企业经营所得

本来应该按"劳务报酬"交20%-45%的个税，通过个体户转成"经营所得"核定征收，实际税负降到几个点。

3. 虚构业务、隐匿收入

部分收入直接不入账，通过关联企业走账隐匿。

为什么翻车了

金税四期 + 大数据比对

- 税务局可以看到你所有银行账户的资金流水

- 个体户的收入来源如果90%以上来自同一个甲方 → 被认定为"实质雇佣关系"，不是经营

- 多个个体户同时给一个人服务 → 明显的拆分避税

- 收入规模和个体户的"经营"规模严重不匹配

你要注意的红线

你的策略五有个体户部分，合法没问题，但要守住这几条线：

- 个体户必须有真实业务，不能只是开票工具

- 客户来源要分散，不能90%收入来自你自己的公司

- 金额要合理，一个"咨询个体户"年开票500万，税务局会盯上

- 不要搞多个个体户，一个就够了，多了就是靶子

- 核定征收是地方政策，随时可能取消，不要把架构建在这上面

案例十三：3个软著换来200万补贴 — 某AI公司的高企认定

真实案例
深圳南山，5人AI公司，做智能客服方案，2023年拿到高企认定

他怎么做到的

时间动作成本
第1年注册3个软件著作权（智能客服系统、语音识别模块、对话管理平台）约600元/个

企税25% → 15%，年省利润的10%

你的情况几乎一模一样

你和这家公司的画像高度重合：5人团队、AI技术公司、有实际产品和营收。

你现在就可以启动：

- 软著登记费约600元/个，30-60天下证（加急更快）

- 同步建研发台账（这个案例八已经讲了）

案例十四：深圳卖家的"自贸区转一圈" — 资金效率提升术

真实案例
深圳跨境电商卖家，年销售额2000万，利润400万

操作方式

在前海注册一家贸易公司，作为"中间商"

工厂（生产）
  ↓ 以成本价+微利 卖给
前海公司（中间商，享受15%企税）
  ↓ 以市场价 卖给
海外客户 / 国内客户

大部分利润留在前海公司（15%企税），而不是留在主体公司（25%企税）。

具体效果

不做架构前海架构
利润归属400万全在主体公司50万在主体（微利）+ 350万在前海
企税400万 × 25% = 100万50万×5% + 350万×15% = 55万
省税—每年省45万

前海的额外好处

- 跨境人民币结算便利（收外汇更方便）

- 保税仓储（货物进出更灵活）

- 港澳专业人士个税补贴（超过15%的部分政府补）

跟你的关系

你的策略一已经规划了南沙/前海子公司，逻辑和这个案例完全一样：

- 越秀母公司做研发（微利或亏损）

- 南沙/前海子公司对外签约收款（利润留这里，享受15%）

- 关键前提：利润超过300万再搞，因为300万以内小微5%比15%还低

- 如果未来有海外客户（AI方案出海），前海的跨境结算优势会很有用

案例十五：程序员辞职创业，第一年就被税"教做人"

常见翻车
某程序员从大厂辞职，接外包单，年入80万，以为跟上班一样交个税就行

他踩了哪些坑

坑1：没注册公司，按"劳务报酬"交税

- 客户要求开发票，他去税务局代开，按"劳务报酬所得"交税

- 80万劳务报酬，扣除20%费用后，适用40%税率，速算扣除7000

- 实际交税约 12万+（还不算增值税附加）

- 如果注册公司（小微），80万利润企税只交4万，差了3倍

坑2：公私不分，所有钱进个人卡

- 客户直接打款到他个人银行卡

- 银行大额交易自动上报，税务局一查就发现收入未申报

- 补税+滞纳金+0.5倍罚款

坑3：不懂季度预缴

- 注册公司后，以为年底一次性报税就行

- 实际上企税要按季度预缴（每季度结束后15天内）

- 漏了几个季度，滞纳金按日万分之五累计

你要检查的

你已经有公司了，但这几个点确认一下：

- 所有业务收入都走公司对公账户，不要走个人卡

- 季度预缴别忘了：每年1/4/7/10月的15号之前

- 给客户开票要及时，收入和开票时间要匹配

- 如果有些小单不方便走公司（比如OPC咨询），以后走个体户

案例十六：京东亏了12年，但刘强东赚翻了 — 亏损的战略价值

高阶思维
京东从2004年到2016年连续亏损12年，但刘强东身价千亿

亏损的"秘密"

账面亏损 ≠ 真的在亏钱

京东每年把利润全部砸进仓储物流基础设施：

- 建仓库 → 固定资产折旧 → 成本↑

- 招物流人员 → 人力成本↑

- 研发物流系统 → 研发费用↑

账面上利润为零甚至亏损，但资产在疯狂增长。

亏损的三大好处

1. 不用交企业所得税（没利润就没税）

2. 亏损可以往后结转5年（高新企业可以10年）。今年亏100万，未来5年赚的钱可以先抵掉这100万再交税

3. 资产在手。仓库、系统、团队都是实实在在的，随时可以变现

和大哥买地是同一个逻辑

利润 → 变成资产 → 账面亏损/微利 → 不交税 → 资产持续升值

区别只是京东买的是仓库和系统，大哥买的是工业用地。

跟你的关系

你现在是成长期，有利润但不多，策略应该是：

- 利润优先投入研发和资产（设备、系统、知识产权），而不是分红

- 如果某一年投入特别大导致亏损，不要慌——亏损可以结转5年抵扣未来利润

- 拿到高企认定后，亏损结转期从5年延长到10年

- 核心思路：账面"穷"，资产"富"
