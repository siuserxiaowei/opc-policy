## 1. 郑东新区支持人工智能OPC创新发展若干措施

- **级别**：区级　**状态**：📢 已发布（细则待公布）
- **发文单位**：郑东新区管委会
- **发布时间**：2026-04-10
- **摘要**：河南省首个OPC专项政策，构建"创业孵化-技术赋能-资本对接-市场拓展"全生命周期服务闭环

**核心支持**：
- 算力服务：依托国家超算郑州中心
- 办公空间："拎脑入驻"模式
- 数据支持：区域内近60家数据交易企业，年交易额44亿元

### 中原科技城OPC生态联盟

- **地址**：郑州中原科技城投资促进中心
- **成立时间**：2026年4月10日
- **参与方**：郑州数据交易中心、曙光信息产业、张江智慧岛（河南）、河南省人工智能协会
- **特色**：拎脑入驻、国家超算郑州中心算力、数据交易生态、全生命周期服务

**参考链接**：[新浪财经](https://finance.sina.com.cn/jjxw/2026-04-11/doc-inhucmte1422501.shtml) | [郑州市科技局](https://zzkj.zhengzhou.gov.cn/zwyw/9889838.jhtml)

## 2. 郑州市支持人工智能创新发展若干政策措施

- **级别**：市级　**状态**：✅ 已在兑现
- **发文单位**：郑州市人民政府（郑政文〔2024〕62号）
- **发布时间**：2024-05-22
- **有效期**：3年（至2027年）

| 补贴项目 | 金额 |
|---|---|
| 算力券补助 | 合同额20%，每企业每年最高 **100万元** |
| 大模型应用场景 | 最高 **500万元** |
| 场景应用创新项目 | 实际投资额30%，最高 **500万元** |
| 重大科技专项 | 单项目不低于 **1000万元** |
| 独角兽企业奖励 | 一次性 **500万元** |
| 潜在独角兽 | 一次性 **200万元** |
| 专精特新小巨人 | 一次性 **100万元** |
| 国家级智能制造示范工厂 | 一次性 **300万元** |

**参考链接**：[郑州市政务公开](https://public.zhengzhou.gov.cn/D0102X/8431444.jhtml) | [郑州市商务局](https://swj.zhengzhou.gov.cn/zcfg/8443549.jhtml)

## 3. 河南省支持人工智能产业生态发展若干政策措施（十条）

- **级别**：省级　**状态**：✅ 已在兑现
- **发文单位**：河南省人民政府（豫政〔2025〕17号）
- **发布时间**：2025-08-04
- **有效期**：3年

| 补贴项目 | 金额 |
|---|---|
| 算力券（个人也可申请） | 费用20%补贴，每企业每年最高 **100万元** |
| 算力券年度总规模 | **5000万元**（省市1:1分担） |
| 国家级大模型备案 | 一次性 **100万元** |
| 行业大模型奖励 | 最高 **100万元** |
| 标杆应用项目 | 最高 **200万元** |
| 行业赋能中心 | 最高 **200万元** |
| 产业基金 | **30亿元**规模 |

> 特别注意：算力券政策明确支持"个人"申请，OPC创业者可以个人身份直接申请算力补贴。

**参考链接**：[河南省发改委](https://fgw.henan.gov.cn/2025/08-09/3200193.html) | [算力券申请入口](https://fgw.henan.gov.cn/2025/03-18/3138097.html)

## 通用创业补贴（郑州OPC创业者可申请）

| 补贴 | 金额 | 条件 |
|---|---|---|
| 一次性创业开业补贴 | 市区1万元 | 首次创办小微企业，正常运营1年以上 |
| 创业担保贷款 | 个人最高30万，小微最高300万 | 财政全额贴息 |
| 大众创业扶持项目 | 省级10-15万，市级最高10万 | 评审通过 |

**申请入口**：[河南就业网上办事大厅](https://hnjy.hrss.henan.gov.cn) | [郑州政务服务网](https://zz.hnzwfw.gov.cn)
