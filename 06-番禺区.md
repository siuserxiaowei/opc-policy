# 番禺区 — 全市首个OPC专项政策

## 《广州市番禺区支持OPC创业若干措施》

**状态**：征求意见稿已发布（2026年3月20日）
**来源**：[南方+·番禺出台全市首个OPC创业扶持政策](https://www.nfnews.com/content/R6gNkMEN3p.html)

---

## 番禺万博CBD OPC空间

| 项目 | 内容 |
|---|---|
| 地址 | 万博商务区、广州大学城 |
| 工位规模 | 1300+ 工位，650+ 人才公寓 |
| 免租政策 | 首年免租，后2年减半 |

## 补贴明细

| 补贴项目 | 金额 |
|---|---|
| 开源贡献/技能包开发补贴 | 最高 **100万元** |
| 创新应用项目奖励（实际投入20%） | 一次性最高 **50万元** |
| 杰出团队发展补贴 | 最高 **500万元** |
| 青年博士落户补贴 | 最高 **10万元** |
| 创业担保贷款 | 最高 **500万元** |

## 其他支持

- 开放政府/国企城市治理、公共服务场景，定期发布"机会清单"
- 支持申领"算力券"
- 国有资本"投早投小投硬科技"，设立OPC产业发展基金
- 聚焦领域：智能制造、智慧政务、智慧园区、智慧医疗
- 入驻方式：番禺区政务服务中心提交
