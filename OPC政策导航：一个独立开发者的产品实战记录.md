# 我为什么给自己做了一个 OPC 政策导航网站

## 起因：政策信息太碎了

2026年3月，广东省发改委发了一份文件——《广东省支持人工智能OPC创新发展行动方案（2026—2028年）》。全国首个省级OPC专项政策，18条措施，目标是到2028年建成100个OPC生态社区，培育1000家标杆企业。

我是一个做AI语音对话机器人的独立开发者，算是OPC超级个体的典型画像。看到这个消息的第一反应是：这些政策我能不能用上？

然后我就开始找。

搜"广州OPC政策"，出来的结果是这样的：
- 公众号文章：标题党居多，"重磅!最高500万!"，点进去发现是把征求意见稿复制粘贴了一遍
- 政府官网：各区各自发布，海珠在海珠的网站，黄埔在黄埔的网站，南沙在南沙的网站
- 新闻报道：散落在21经济网、南方财经、南方日报、新浪财经……每篇报道角度不一样，数据不一致
- 创业公众号：把各种补贴东拼西凑，金额和条件经常对不上

我花了差不多两天时间，打开了几十个网页，才把广州各区的OPC政策大致理清楚。但我还是不确定：**到底哪些政策我能申请？需要准备什么材料？哪些有截止日期？**

这就是问题所在——OPC政策本身是好的，但**信息获取的成本太高了**。

## 什么是OPC？为什么它值得关注？

OPC，One Person Company，一人公司。但它不只是"一个人开公司"这么简单。

在AI工具爆发的今天，一个人可以做到：
- 用Claude/GPT写代码、做产品
- 用Midjourney/Flux做设计
- 用AI语音对话机器人做客服
- 用自动化工具做营销获客

以前需要10人团队才能做的事，现在一个人+AI就能搞定。这就是"超级个体"。

而OPC政策，本质上是政府在用真金白银鼓励这种创业模式。

### 这不是个别城市的试水，是全国性的趋势

广东省带头出了省级方案，然后像多米诺骨牌一样，各地政府纷纷跟进。截至2026年4月，我统计到**37个城市、81条政策、130+个OPC社区**。

力度最大的几个城市：

**广州**——政策条目最多（12条），各区内卷严重。海珠区出了"OPC十条"，场景征集项目最高补贴500万；黄埔区鱼珠·元启社区免租3年；南沙区0.5天拿照，号称全国最快注册。番禺区出了全市首个OPC专项政策，开源代码贡献奖最高100万。

**深圳**——生态最完善。2026年1月深圳市工信局就发了《打造人工智能OPC创业生态引领地行动计划》，目标到2027年底建成10+家OPC社区（每家1万㎡以上），培育1000+家AI创业企业。深圳的玩法很成体系——"一张桌、一间办公室、一层楼"三级空间体系，按团队规模匹配。资金支持也猛：鲲鹏青年项目最高100万研发资助，留学回国青年最高100万创业资助，"训力券"最高1000万、"模型券""语料券"各200万。各区还在加码——罗湖区出了10大措施，算力补贴年度100万，股权跟投最高1000万；南山区"模力营"已经入驻近200家企业，超10万㎡，大湾区最大；龙岗区"龙虾十条"智能体补贴最高200万。注册也很方便，通过"i深圳"APP一站式办理，1个工作日办结。

**苏州**——政策数量全国第一（14条），多个区都有独立的OPC扶持措施，算力券、孵化空间、创业资助一应俱全。

**北京**——算力补贴最高（2000万），中关村有OPC社区，但门槛相对较高。

**上海**——场景示范最高1000万，张江有OPC社区，侧重AI应用落地。

### 主要支持方向

不管哪个城市，OPC政策的核心支持基本围绕几个方向：

- **算力补贴**：用大模型跑推理要烧算力，政府帮你出一半甚至更多的钱。深圳的"训力券"最高给1000万
- **免费办公空间**：琶洲模方、鱼珠元启、南沙国际社区、深圳模力营……免租1-3年不是梦
- **创业资金**：一次性资助1万起步，租金补贴每年6000，创业担保贷款最高500万（深圳个人60万、小微500万）
- **场景开放**：政府把医疗、交通、教育、政务等场景开放给你做AI应用，项目补贴几十万到几百万
- **人才补贴**：本科/硕士/博士入户补贴，人才公寓，安居补贴
- **快速注册**：南沙0.5天拿照，深圳"i深圳"APP一站式1日办结

这些不是画饼，是真金白银。但前提是——**你得知道有这些政策，而且知道怎么申请**。

## 所以我自己做了一个

既然没有好的工具，那就自己造一个。

第一版很简单：把我整理好的广州OPC政策写成一份结构化的Markdown文档，按区域分类，每条政策标注来源链接、补贴金额、申请条件。然后放到GitHub上，用GitHub Pages部署成一个静态网站。

然后逐步迭代：

### v1: 广州政策文档

最开始只是一份汇总文档，逐条人工验证了所有链接和数据。验证方式很朴素——每个链接都点开看，每个金额都去政府原文核对。

一个重要教训：**最早的那版数据是AI（Codex）生成的，有不少幻觉**——编造的链接、虚构的金额、不存在的政策。后来我用全网搜索交叉验证，17个链接逐一检查，才把假信息清理干净。这件事让我对AI生成的信息保持了高度警惕。

### v2: 结构化数据库 + 用户匹配

手动整理太累了。我把每条政策拆分成标准的JSON字段：城市、级别、发文单位、补贴类型、金额上限、申请条件、截止日期……做成一个 `policies.json` 数据库。

然后做了一个匹配系统：用户填写自己的公司情况（在哪个城市注册、成立多久、有几个人缴社保、做什么行业），系统自动算分，匹配出你能申请的政策。

### v3: 扩展到全国

广州做完了，但其他城市呢？苏州OPC政策数量全国第一（14条），深圳有7条且生态最完善，北京上海各5条。我把数据库扩展到了37个城市、81条政策、130+个OPC社区。

每个城市单独生成政策文档，放在GitHub上供查阅。

### v4: AI政策顾问 + 数据看板 + 材料清单

三个实用功能：
- **AI政策顾问**：在网页里直接问"我在深圳做AI语音机器人，有什么政策？"，本地匹配37城数据回答
- **数据看板**：全国政策数量、补贴排行、城市对比、省份分布，一眼看清
- **材料清单生成器**：匹配到政策后，自动生成你需要准备的材料checklist，可勾选已备齐的

### v5: 自动爬虫 + 政策变动追踪

政策是会变的。征求意见稿会变成正式文件，截止日期会到期，新政策会发布。靠人工刷新不现实。

做了一个Cloudflare Worker爬虫，每12小时自动扫描44个政府网站（覆盖广州各区、深圳、苏州、北京、上海等），用22个关键词（OPC、一人公司、超级个体、算力补贴、创业资助、智能体、大模型……）匹配相关链接，然后用Google Gemini 2.0 Flash提取结构化政策数据。

同时做了政策变动追踪：自动检测新增政策、状态变更（征求意见→正式实施）、过期政策。

发现新政策后，用户下次刷新页面就能看到——整条链路全自动。

整套系统的运行成本：**$0/月**。

Cloudflare Workers免费，KV存储免费，Gemini用Google送的$300额度（够跑200万次）。一个人、零成本、全自动。

### v6: 税务指南

注册公司只是第一步，接下来最头疼的就是税务问题。个人独资企业、一人有限公司、个体工商户，到底选哪种？核定征收还是查账征收？前海/南沙/海南的15%企税优惠怎么用？

我做了一个税负计算器，输入年营收和利润率，自动对比三种主体的综合税负，告诉你哪个最优。

这块我请了一个专业的税务顾问——张永航老师（盈丰税务规划），多年财税从业经验，专门帮OPC创业者做财税合规和税务优化。税务问题专业性太强，该找专业人的时候得找专业人。

## 技术栈

刻意做得极简：

```
前端：纯HTML + CSS + 原生JS（无框架，单文件）
数据：JSON文件（GitHub托管）
部署：GitHub Pages + Cloudflare Pages（双线）
爬虫：Cloudflare Workers（免费）
AI提取：Google Gemini 2.0 Flash（$300额度）
域名：opcgate.com
```

不用React，不用数据库，不用后端服务器。一个人能维护的系统才是好系统。

## 一些想法

### 先为自己服务

这个项目的出发点不是"做一个产品去融资"，而是"我自己需要这个东西"。我是一个OPC创业者，我需要知道哪些政策能帮我省钱、拿补贴。

做出来之后发现，这个需求是普遍的。每个OPC创业者都面临同样的信息获取困境。

### 政策信息应该是透明的

政府花了大力气制定OPC支持政策，但如果创业者根本不知道这些政策的存在，或者搞不清自己能不能申请，那政策的效果就大打折扣。

以深圳为例，市级+区级政策叠加，罗湖的股权跟投最高1000万，南山的模力营已经聚集了近200家企业。这些信息散落在gxj.sz.gov.cn、sz.gov.cn、各区政府网站上，普通创业者很难系统性地获取。

我做的这个工具，本质上是在帮政策更好地触达创业者。

### OPC是认真的

有人觉得OPC是一阵风，过两年就凉了。我不这么看。

当一个省级政府发布专项行动方案，深圳要建10+个万平方米级社区，苏州一口气出14条政策，广州各区为了抢OPC企业内卷到飞起——这不是试水，这是基础设施建设。

AI正在改变工作的基本单元。以前是团队，现在可以是个人。OPC政策是在为这个变化做制度准备。

现在是最佳窗口期。政策刚出、补贴充足、审批宽松。等到所有人都知道了，名额满了，门槛提高了，就晚了。
